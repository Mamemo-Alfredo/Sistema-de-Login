# 🔐 Sistema de Login com JavaScript

Aplicação simples de autenticação desenvolvida com **HTML, CSS e JavaScript**, utilizando **LocalStorage** para persistência de dados no navegador.

---

## 📌 Sobre o Projeto

Este projeto demonstra um sistema básico de login e registro de usuários no lado do cliente (frontend), sem utilização de backend.

Os dados são armazenados localmente no navegador através do **LocalStorage**, permitindo simular autenticação de usuários.

---

## 🚀 Tecnologias Utilizadas

* 🌐 HTML5
* 🎨 CSS3
* ⚡ JavaScript
* 💾 LocalStorage

---

## 🎯 Funcionalidades

* 📝 Cadastro de utilizador
* 🔑 Login com validação de credenciais
* 💾 Armazenamento de dados no LocalStorage
* 🚪 Logout do sistema
* 🔒 Controle básico de acesso (simulado)
* 📱 Interface simples e responsiva

---

## ⚙️ Como Funciona

* Os dados do utilizadores são salvos no **LocalStorage**
* No login, o sistema compara os dados inseridos com os armazenados
* Se forem válidos, o utilizadores é autenticado
* O estado de login pode ser mantido no navegador

---

## ⚠️ Limitações

Este projeto é apenas para fins educacionais.

* ❌ Não possui criptografia de senha
* ❌ Não possui backend
* ❌ Não é seguro para uso em produção

---

## 💡 Objetivo

O objetivo deste projeto é praticar:

* Manipulação do DOM
* Validação de formulários
* Armazenamento de dados no navegador
* Lógica de autenticação
